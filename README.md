# Seminar_Management_System

# Project Description
Developed a Java-based parking management system to handle vehicle entry, exit, and parking records, with automated tracking of parking duration and fee calculation.

# Key Features
-Built a console-based application for managing parking operations
-Implemented vehicle check-in and check-out system
-Recorded and managed:
  Vehicle number and type
  Entry and exit timestamps
  Parking duration
  Developed logic to calculate parking fees based on time spent
-Utilized file handling (File I/O) to store and retrieve parking records
-Ensured data persistence without using a database

# Tools & Concepts
-Java
-Object-Oriented Programming (OOP)
